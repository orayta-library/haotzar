# PDF Indexer - Standalone Package

## What's included?

This package includes:
- All Python source code
- All required libraries (wheels)
- Installation scripts
- Documentation

## Installation (Offline)

1. Make sure Python 3.8+ is installed on the target computer
2. Run: `install_offline.bat`
3. Wait for installation to complete

## Usage

1. Run: `run_ui.bat`
2. Open browser: http://localhost:5000

## Requirements

- Python 3.8 or higher
- Windows OS
- No internet connection needed

## File Structure

```
pdf-indexer-standalone/
├── wheels/              (All Python packages)
├── templates/           (HTML files)
├── *.py                (Source code)
├── install_offline.bat (Offline installer)
├── run_ui.bat          (Run script)
└── README.md           (This file)
```

## Troubleshooting

### Python not found
Install Python from: https://www.python.org/downloads/
Make sure to check "Add Python to PATH" during installation.

### Installation fails
Make sure you have write permissions in the directory.
Try running as Administrator.
