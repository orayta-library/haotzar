# 🎨 מדריך שימוש ב-Web UI

## 🚀 הפעלה מהירה

### 1. התקנה (פעם אחת)
```bash
cd pdf-indexer
.\install.bat
```

### 2. הפעל את השרת
```bash
.\run_ui.bat
```

### 3. פתח בדפדפן
```
http://localhost:5000
```

## 📱 ממשק המשתמש

### מסך ראשי

![UI Screenshot](screenshot.png)

הממשק כולל:
- 📂 **בחירת תיקייה** - בחר את תיקיית הספרים
- ⚙️ **הגדרות** - התאם אישית את תהליך הבנייה
- 📊 **התקדמות** - מעקב בזמן אמת
- ⚠️ **שגיאות** - הצגת שגיאות אם יש

## 🎯 תכונות עיקריות

### 1. בחירת תיקייה
- לחץ על כפתור "📁 עיון"
- הזן נתיב ידנית או עיין בתיקיות
- התיקייה תישמר להפעלה הבאה

### 2. זיהוי Checkpoint אוטומטי

אם יש checkpoint קיים, תראה:

```
📍 נמצא checkpoint קיים
50 קבצים עובדו | 150 נותרו | 25% הושלם
```

אפשרויות:
- **המשך** - המשך מאיפה שעצרת (ברירת מחדל)
- **התחל מחדש** - סמן "התחל מחדש (מחק checkpoint)"

### 3. הגדרות מתקדמות

#### Flush כל (קבצים)
- **1** - חיסכון מקסימלי בזיכרון (איטי יותר)
- **2** - איזון (מומלץ) ✅
- **5** - מהיר יותר (צורך יותר זיכרון)

#### העלה ל-Meilisearch
- סמן אם אתה רוצה להעלות ישירות ל-Meilisearch
- ודא ש-Meilisearch רץ לפני!

### 4. מעקב בזמן אמת

במהלך הבנייה תראה:

```
📊 התקדמות
[████████████████░░░░] 75%

קבצים שעובדו: 150 / 200
Chunks: 45,231
מילים ייחודיות: 125,847
זמן שחלף: 03:45

מעבד כעת:
משנה ברורה - חלק ראשון.pdf
```

### 5. עצירה בטוחה

לחץ על "⏹️ עצור" בכל עת:
- התהליך יעצור בצורה מסודרת
- ה-checkpoint יישמר
- תוכל להמשיך מאוחר יותר

## 📊 דוגמאות שימוש

### דוגמה 1: בניית אינדקס ראשונה

1. פתח http://localhost:5000
2. הזן תיקייה: `D:\books`
3. השאר הגדרות ברירת מחדל
4. לחץ "🚀 התחל בניית אינדקס"
5. המתן עד לסיום

### דוגמה 2: המשך אחרי הפסקה

1. פתח http://localhost:5000
2. תראה: "📍 נמצא checkpoint קיים"
3. לחץ "🚀 התחל בניית אינדקס"
4. התהליך ימשיך מאיפה שעצר

### דוגמה 3: בניה מחדש

1. פתח http://localhost:5000
2. סמן "התחל מחדש (מחק checkpoint)"
3. לחץ "🚀 התחל בניית אינדקס"
4. התהליך יתחיל מאפס

### דוגמה 4: בניה + העלאה ל-Meilisearch

1. הפעל Meilisearch:
   ```bash
   ..\resources\meilisearch\meilisearch.exe
   ```
2. פתח http://localhost:5000
3. סמן "העלה ל-Meilisearch"
4. לחץ "🚀 התחל בניית אינדקס"
5. בסיום, הנתונים יועלו אוטומטית

## 🔧 פתרון בעיות

### השרת לא עולה
```bash
# בדוק שאין שרת אחר על פורט 5000
netstat -ano | findstr :5000

# אם יש, עצור אותו או שנה פורט ב-web_ui.py
```

### לא רואה את התיקייה
- ודא שהנתיב נכון
- השתמש ב-`\` או `/` בנתיב
- דוגמה: `D:\books` או `D:/books`

### Checkpoint לא נשמר
- ודא שיש הרשאות כתיבה לתיקיית הפלט
- בדוק שיש מספיק מקום בדיסק

### Meilisearch לא עובד
```bash
# ודא ש-Meilisearch רץ
curl http://127.0.0.1:7700/health

# אם לא, הפעל:
..\resources\meilisearch\meilisearch.exe
```

## 💡 טיפים

1. **השתמש ב-SSD** - שים את הספרים ב-SSD למהירות מקסימלית
2. **סגור תוכנות** - סגור תוכנות אחרות בזמן הבנייה
3. **עקוב אחרי התקדמות** - השאר את הדפדפן פתוח
4. **שמור checkpoint** - לחץ "עצור" במקום לסגור את החלון
5. **בדוק שגיאות** - אם יש שגיאות, הן יופיעו למטה

## 🚀 שימוש מתקדם

### הרצה על פורט אחר
ערוך `web_ui.py`:
```python
app.run(host='0.0.0.0', port=8080, debug=False)
```

### גישה מרחוק
```python
# web_ui.py
app.run(host='0.0.0.0', port=5000, debug=False)
```

אז תוכל לגשת מכל מחשב ברשת:
```
http://192.168.1.100:5000
```

### הרצה כ-service (Windows)
השתמש ב-NSSM או Task Scheduler להרצה אוטומטית.
