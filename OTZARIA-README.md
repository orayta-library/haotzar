# הוראות שימוש - אינטגרציה של אוצריא

## מה נוסף?

התוכנה שלך תומכת כעת בקריאה ישירה ממסד הנתונים של אוצריא! 🎉

הספרים מאוצריא יופיעו בספרייה הרגילה תחת תיקייה בשם **"אוצריא"** עם כל הקטגוריות והספרים בהיררכיה מסודרת.

## ⚠️ לפני שמתחילים - תיקון חובה!

אם זו ההפעלה הראשונה, **חובה** להריץ:

### אופציה 1: סקריפט אוטומטי (קל ביותר)
לחץ פעמיים על הקובץ:
```
fix-sqlite.cmd
```

### אופציה 2: ידני
פתח טרמינל והרץ:
```bash
npm rebuild better-sqlite3
```

**למה זה נדרש?** 
`better-sqlite3` צריך להיות מקומפל לגרסת Node.js הנכונה.

---

## איך להתחיל?

### שלב 1: הכנת קובץ ה-DB

1. וודא שיש לך את קובץ `seforim.db` של אוצריא
2. העתק אותו לתיקייה:
   ```
   books/אוצריא/seforim.db
   ```

### שלב 2: הרץ את האפליקציה

```bash
npm run electron:dev
```

או אם אתה בונה גרסת production:
```bash
npm run electron:build
```

### שלב 3: פתח את הספרייה

1. לחץ על כפתור הספרייה בצד שמאל
2. תראה תיקייה חדשה בשם **"אוצריא"**
3. פתח אותה ותראה את כל הקטגוריות:
   - תנ"ך
   - משנה
   - תלמוד
   - מדרש
   - הלכה
   - ועוד...

### שלב 4: פתח ספר

1. נווט בעץ הקטגוריות
2. לחץ על ספר כלשהו
3. הספר ייפתח בכרטיסייה חדשה עם:
   - תוכן עניינים בצד (אם קיים)
   - הטקסט המלא
   - ניווט בין עמודים

## תכונות

✅ **הצגה היררכית** - כל הקטגוריות והספרים מסודרים בעץ  
✅ **תוכן עניינים** - ניווט מהיר בתוך הספר  
✅ **פגינציה** - טעינה מהירה (100 שורות בכל פעם)  
✅ **אינטגרציה מלאה** - עובד בדיוק כמו PDF/TXT רגילים  
✅ **כרטיסיות מרובות** - פתח כמה ספרים בו-זמנית  
✅ **Split View** - הצג שני ספרים זה לצד זה  

## טיפים

- **חיפוש מהיר**: השתמש בחיפוש בספרייה כדי למצוא ספרים
- **תוכן עניינים**: לחץ על פריט בתוכן העניינים כדי לקפוץ לשורה
- **ניווט**: השתמש בכפתורי "הקודם"/"הבא" למעבר בין עמודים

## בעיות?

### תיקיית "אוצריא" לא מופיעה

1. בדוק שהקובץ נמצא ב: `books/אוצריא/seforim.db`
2. בדוק את הקונסול (F12) לשגיאות
3. וודא שהקובץ לא פגום (גודל > 6GB)
4. **הכי חשוב**: הרץ `npm rebuild better-sqlite3`

### שגיאת NODE_MODULE_VERSION

אם אתה רואה שגיאה כזו:
```
NODE_MODULE_VERSION 141 vs 143
```

**פתרון:**
```bash
npm rebuild better-sqlite3
```

או הרץ את `fix-sqlite.cmd`

### עדיין לא עובד?

ראה את הקובץ `FIX-OTZARIA.md` לפתרונות נוספים.
